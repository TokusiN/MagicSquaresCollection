# MagicSquaresCollection

高度な探索アルゴリズムと計算機パワーによって発見された、特殊な多重魔方陣・魔方陣のコレクションです。

## 収録コンテンツ概要

### 1. Add-Mult Magic Cube 32x32x32
32次の立体和積魔方陣です。
- **特徴**: 全ての行・列・柱・立体対角線の「和」と「積」が一致。
- **手法**: XORベースのビット配置、(n+1),(32n+1),(255n+1)の直積

### 2. Add-Mult-Mult1 Magic Square 512x512
512次の非常に特殊な多重魔方陣です。
- **特徴**: 
    - **Add-Mult**: 全ての行・列・対角線の「和」と「積」が一致。
    - **Mult1**: n^nの積が一致（k乗和のk=1が重根になること及び、n log nの和が一致することと同値）
- **手法**: XORベースのビット配置、(n+1),(64n+1),(660n+1)の直積

### 3. Pentamagic Square 512x512
512次の正規五重方陣です。
- **特徴**: 全ての行・列・対角線の、 $S\_1$ から $S\_5$ （1乗和から5乗和）が一致。
- **手法**: XORベースのビット配置により対称な水平5重・垂直4重方陣を構築。
  その後、対称位置の要素置換と4段のカスケード式中間一致攻撃により、5乗和を調整

### 4. Tetramagic Square 64x64
64次の正規四重方陣です。
- **特徴**: 全ての行・列・対角線の、 $S\_1$ から $S\_4$ （1乗和から4乗和）が一致。
- **手法**: 対称な数を同じセルに配置した32x64のTetramagic squreをナップザック問題の探索により作成。
  その後、S1,S3を崩さないように対称位置の数字を左右に振り分け

### 5. Tetramagic Square 60x60
60次の正規四重方陣です。
- **特徴**: 全ての行・列・対角線の、 $S\_1$ から $S\_4$ （1乗和から4乗和）が一致。
- **手法**: 64次と同じ。2の冪以外、特に8n+4のサイズのTetramagic Squareが存在する事を確かめるために作成。

### 6. Tetramagic Square 56x56
56次の正規四重方陣です。
- **特徴**: 全ての行・列・対角線の、 $S\_1$ から $S\_4$ （1乗和から4乗和）が一致。
- **手法**: 64次と同じ。記録更新および、今のアルゴリズムの限界を探るために作成。

### 7. Tetramagic Square 52x52
52次の正規四重方陣です。
- **特徴**: 全ての行・列・対角線の、 $S\_1$ から $S\_4$ （1乗和から4乗和）が一致。
- **手法**: 64次と同じ。記録更新および、今のアルゴリズムの限界を探るために作成。

### 8. Tetramagic Square 48x48
48次の正規四重方陣です。
- **特徴**: 全ての行・列・対角線の、 $S\_1$ から $S\_4$ （1乗和から4乗和）が一致。
- **手法**: 64次と同じ。これより小さいものは現在のアルゴリズムでは困難と判断したため、Tetramagic Squareの探索は一旦これで終了。

### 9. Bimagic Square of Squares 40x40
40次の2乗数で作られた二重方陣、つまり$S\_2$ と $S\_4$のみが成立する魔方陣です。
- **特徴**: 全ての行・列・対角線の、 $S\_2$ と $S\_4$ （2乗和と4乗和）が一致。
- **手法**: 横列、縦列、対角線の順に中間一致攻撃。
  例えば一番上の列に225～232の連続数が含まれているように、横列作成時に全ての行に8個の連続数を仕込みました。
  これを縦列の最後の調整の時に使用しています。

## 検証について
各ファイルに収録された魔方陣は、スクリプトにより以下の項目が検証されています。
- 縦・横・斜めの各累乗和（ $S\_1$ ～ $S\_n$ ）・積などの一致。
- 全要素の重複なし
